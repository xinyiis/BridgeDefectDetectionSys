## 1. 业务需求分析

### 1.1 核心功能

报表生成模块是桥梁缺陷检测系统的重要输出功能，为用户提供专业的PDF检测报告。

**主要功能**：
1. **生成检测报告**：根据桥梁和时间范围生成PDF报告
2. **报表管理**：查询历史报表、下载报表
3. **数据可视化**：缺陷统计图表、趋势分析图表
4. **报表归档**：持久化存储，支持历史查询

### 1.2 业务场景

#### 场景1：桥梁定期检测报告
- 用户：桥梁管理员
- 操作：生成某座桥梁过去30天的检测报告
- 内容：缺陷统计、缺陷类型分布、健康度评分、缺陷详情列表、缺陷图片
- 用途：向上级汇报、存档备查

#### 场景2：缺陷专项分析报告
- 用户：技术人员
- 操作：生成某座桥梁某段时间内的高危缺陷报告
- 内容：高危缺陷列表、位置分布、严重程度分析、修复建议
- 用途：制定维修计划

#### 场景3：多桥梁对比报告
- 用户：管理员
- 操作：生成多座桥梁的健康度对比报告
- 内容：桥梁健康度排名、缺陷数量对比、趋势对比
- 用途：资源分配、优先级决策

### 1.3 报表类型

根据接口文档和业务需求，设计3种报表类型：

| 报表类型 | 描述 | 数据来源 |
|---------|------|---------|
| **桥梁检测报告** | 单座桥梁的全面检测报告 | 桥梁信息+缺陷记录+统计数据 |
| **缺陷分析报告** | 某时间段的缺陷专项分析 | 缺陷记录+统计分析 |
| **健康度对比报告** | 多座桥梁的健康度对比 | 桥梁列表+健康度统计 |

### 1.4 报表内容

#### 桥梁检测报告（主要类型）

**报表封面**：
- 报表标题
- 桥梁名称
- 报表时间范围
- 生成日期
- 报表编号

**第一部分：桥梁基本信息**
- 桥梁名称、编号、地址
- 桥梁类型、建造年份
- 长度、宽度
- 3D模型缩略图（可选）

**第二部分：检测概览**
- 检测时间范围
- 检测次数
- 缺陷总数
- 新增缺陷数
- 高危缺陷数
- 健康度评分（大号显示）

**第三部分：缺陷统计分析**
- 缺陷类型饼图（裂缝、剥落、钢筋锈蚀等）
- 缺陷趋势折线图（按日期统计）
- 缺陷严重程度分布柱状图
- 缺陷位置热力图（可选）

**第四部分：高危缺陷列表**
- 表格展示（序号、缺陷类型、位置、面积、置信度、检测时间）
- 前10个高危缺陷的图片展示（2列布局）

**第五部分：缺陷详细列表**
- 按缺陷类型分组
- 每组显示：类型名称、数量、平均面积、平均置信度
- 详细表格（缩略版）

**第六部分：建议和结论**
- 健康度评估
- 风险提示
- 维护建议
- 下次检测建议时间

**报表页脚**：
- 页码（第X页 / 共Y页）
- 生成时间
- 系统版权信息

## 2. 技术方案选型

### 2.1 PDF生成库选择

#### 方案1：gofpdf（推荐）

**优点**：
- 纯Go实现，无需外部依赖
- 轻量级，性能好
- 支持中文（需要字体文件）
- 支持图片、表格、多列布局
- API简单易用

**缺点**：
- 需要手动布局
- 图表需要先生成图片

**示例代码**：
```go
pdf := gofpdf.New("P", "mm", "A4", "")
pdf.AddPage()
pdf.SetFont("Arial", "B", 16)
pdf.Cell(40, 10, "Hello World")
pdf.OutputFileAndClose("report.pdf")
```

#### 方案2：wkhtmltopdf（备选）

**优点**：
- HTML转PDF，样式丰富
- 支持CSS布局
- 前端人员友好

**缺点**：
- 需要外部可执行文件
- 性能较低
- 部署复杂

**结论**：选择 **gofpdf**，理由：
1. 纯Go实现，部署简单
2. 性能好，适合批量生成
3. 与现有Go技术栈一致

### 2.2 图表生成方案

#### 方案1：go-echarts + Chrome Headless

**优点**：
- ECharts图表效果好
- 支持复杂图表类型

**缺点**：
- 需要Chrome/Chromium
- 部署复杂

#### 方案2：go-chart（推荐）

**优点**：
- 纯Go实现
- 支持常用图表（柱状图、折线图、饼图）
- 直接生成PNG图片
- 无外部依赖

**缺点**：
- 图表样式相对简单

**示例代码**：
```go
import "github.com/wcharczuk/go-chart/v2"

graph := chart.PieChart{
    Values: []chart.Value{
        {Value: 5, Label: "裂缝"},
        {Value: 3, Label: "剥落"},
    },
}
buffer := bytes.NewBuffer([]byte{})
graph.Render(chart.PNG, buffer)
```

**结论**：选择 **go-chart**，理由：
1. 纯Go实现，部署简单
2. 满足基本图表需求
3. 与gofpdf集成方便

### 2.3 技术栈总结

| 技术 | 库/工具 | 用途 |
|------|--------|------|
| PDF生成 | `github.com/jung-kurt/gofpdf` | 生成PDF文件 |
| 图表生成 | `github.com/wcharczuk/go-chart/v2` | 生成图表PNG |
| 中文字体 | 思源黑体 (Source Han Sans) | 支持中文显示 |
| 图片处理 | `image` 标准库 | 图片缩放、裁剪 |
| 时间处理 | `time` 标准库 | 日期格式化 |

## 3. 数据库设计

### 3.1 报表表（reports）

```sql
CREATE TABLE reports (
    id BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY COMMENT '报表ID',
    report_name VARCHAR(200) NOT NULL COMMENT '报表名称',
    report_type ENUM('bridge_inspection', 'defect_analysis', 'health_comparison') NOT NULL COMMENT '报表类型',
    user_id BIGINT UNSIGNED NOT NULL COMMENT '创建用户ID',
    bridge_id BIGINT UNSIGNED COMMENT '关联桥梁ID（单桥梁报表）',
    bridge_ids JSON COMMENT '关联桥梁ID列表（多桥梁报表）',
    start_time DATETIME NOT NULL COMMENT '报表开始时间',
    end_time DATETIME NOT NULL COMMENT '报表结束时间',
    file_path VARCHAR(500) COMMENT 'PDF文件路径',
    file_size BIGINT COMMENT '文件大小（字节）',
    status ENUM('generating', 'completed', 'failed') DEFAULT 'generating' COMMENT '生成状态',
    error_message TEXT COMMENT '错误信息（失败时）',
    total_pages INT COMMENT '总页数',
    defect_count INT COMMENT '缺陷数量',
    high_risk_count INT COMMENT '高危缺陷数量',
    health_score FLOAT COMMENT '健康度评分',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    deleted_at TIMESTAMP NULL COMMENT '删除时间（软删除）',

    INDEX idx_user_id (user_id),
    INDEX idx_bridge_id (bridge_id),
    INDEX idx_report_type (report_type),
    INDEX idx_status (status),
    INDEX idx_created_at (created_at),
    INDEX idx_deleted_at (deleted_at),

    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (bridge_id) REFERENCES bridges(id) ON DELETE SET NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='检测报表';
```

### 3.2 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| report_type | ENUM | bridge_inspection（桥梁检测）/ defect_analysis（缺陷分析）/ health_comparison（健康对比） |
| bridge_ids | JSON | 多桥梁报表时存储桥梁ID数组：`[1,2,3]` |
| status | ENUM | generating（生成中）/ completed（已完成）/ failed（失败） |
| file_size | BIGINT | 用于文件管理和清理 |
| total_pages | INT | 用于快速展示报表信息 |

### 3.3 索引设计

1. **idx_user_id**: 用户查询自己的报表
2. **idx_bridge_id**: 按桥梁查询报表
3. **idx_report_type**: 按类型筛选
4. **idx_status**: 查询生成中/失败的报表
5. **idx_created_at**: 按时间排序
6. **idx_deleted_at**: 软删除过滤

```go
package model

import (
    "database/sql/driver"
    "encoding/json"
    "time"
)

// ReportType 报表类型
type ReportType string

const (
    ReportTypeBridgeInspection  ReportType = "bridge_inspection"  // 桥梁检测报告
    ReportTypeDefectAnalysis    ReportType = "defect_analysis"    // 缺陷分析报告
    ReportTypeHealthComparison  ReportType = "health_comparison"  // 健康对比报告
)

// ReportStatus 报表状态
type ReportStatus string

const (
    ReportStatusGenerating ReportStatus = "generating" // 生成中
    ReportStatusCompleted  ReportStatus = "completed"  // 已完成
    ReportStatusFailed     ReportStatus = "failed"     // 失败
)

// BridgeIDs 桥梁ID列表（JSON字段）
type BridgeIDs []uint

// Scan 实现sql.Scanner接口
func (b *BridgeIDs) Scan(value interface{}) error {
    if value == nil {
        *b = nil
        return nil
    }
    bytes, ok := value.([]byte)
    if !ok {
        return errors.New("type assertion to []byte failed")
    }
    return json.Unmarshal(bytes, b)
}

// Value 实现driver.Valuer接口
func (b BridgeIDs) Value() (driver.Value, error) {
    if len(b) == 0 {
        return nil, nil
    }
    return json.Marshal(b)
}

// Report 报表模型
type Report struct {
    ID             uint          `gorm:"primaryKey" json:"id"`
    ReportName     string        `gorm:"size:200;not null" json:"report_name"`
    ReportType     ReportType    `gorm:"type:enum('bridge_inspection','defect_analysis','health_comparison');not null" json:"report_type"`
    UserID         uint          `gorm:"not null;index" json:"user_id"`
    BridgeID       *uint         `gorm:"index" json:"bridge_id"`                  // 单桥梁报表
    BridgeIDs      BridgeIDs     `gorm:"type:json" json:"bridge_ids"`            // 多桥梁报表
    StartTime      time.Time     `gorm:"not null" json:"start_time"`
    EndTime        time.Time     `gorm:"not null" json:"end_time"`
    FilePath       string        `gorm:"size:500" json:"file_path"`
    FileSize       int64         `json:"file_size"`
    Status         ReportStatus  `gorm:"type:enum('generating','completed','failed');default:'generating';index" json:"status"`
    ErrorMessage   string        `gorm:"type:text" json:"error_message,omitempty"`
    TotalPages     int           `json:"total_pages"`
    DefectCount    int           `json:"defect_count"`
    HighRiskCount  int           `json:"high_risk_count"`
    HealthScore    float64       `json:"health_score"`
    CreatedAt      time.Time     `gorm:"index" json:"created_at"`
    UpdatedAt      time.Time     `json:"updated_at"`
    DeletedAt      *time.Time    `gorm:"index" json:"deleted_at,omitempty"`

    // 关联
    User   *User   `gorm:"foreignKey:UserID" json:"user,omitempty"`
    Bridge *Bridge `gorm:"foreignKey:BridgeID" json:"bridge,omitempty"`
}

// TableName 指定表名
func (Report) TableName() string {
    return "reports"
}

// IsOwnedBy 检查报表是否属于指定用户
func (r *Report) IsOwnedBy(userID uint) bool {
    return r.UserID == userID
}

// IsCompleted 检查报表是否生成完成
func (r *Report) IsCompleted() bool {
    return r.Status == ReportStatusCompleted
}

// IsFailed 检查报表是否生成失败
func (r *Report) IsFailed() bool {
    return r.Status == ReportStatusFailed
}

// MarkAsCompleted 标记报表为已完成
func (r *Report) MarkAsCompleted(filePath string, fileSize int64, totalPages int) {
    r.Status = ReportStatusCompleted
    r.FilePath = filePath
    r.FileSize = fileSize
    r.TotalPages = totalPages
    r.ErrorMessage = ""
}

// MarkAsFailed 标记报表为失败
func (r *Report) MarkAsFailed(errorMsg string) {
    r.Status = ReportStatusFailed
    r.ErrorMessage = errorMsg
}
```

```go
package persistence

import (
    "errors"
    "gorm.io/gorm"
    "github.com/xinyiis/BridgeDefectDetectionSys/internal/domain/model"
    "github.com/xinyiis/BridgeDefectDetectionSys/internal/domain/repository"
)

type ReportRepositoryImpl struct {
    db *gorm.DB
}

func NewReportRepository(db *gorm.DB) repository.ReportRepository {
    return &ReportRepositoryImpl{db: db}
}

func (r *ReportRepositoryImpl) Create(report *model.Report) error {
    return r.db.Create(report).Error
}

func (r *ReportRepositoryImpl) FindByID(id uint) (*model.Report, error) {
    var report model.Report
    err := r.db.Preload("User").Preload("Bridge").First(&report, id).Error
    if err != nil {
        if errors.Is(err, gorm.ErrRecordNotFound) {
            return nil, errors.New("报表不存在")
        }
        return nil, err
    }
    return &report, nil
}

func (r *ReportRepositoryImpl) Update(report *model.Report) error {
    return r.db.Save(report).Error
}

func (r *ReportRepositoryImpl) Delete(id uint) error {
    // 软删除
    return r.db.Delete(&model.Report{}, id).Error
}

func (r *ReportRepositoryImpl) List(page, pageSize int, reportType *model.ReportType) ([]model.Report, int64, error) {
    var reports []model.Report
    var total int64

    query := r.db.Model(&model.Report{}).Preload("Bridge")

    // 类型过滤
    if reportType != nil && *reportType != "" {
        query = query.Where("report_type = ?", *reportType)
    }

    // 统计总数
    if err := query.Count(&total).Error; err != nil {
        return nil, 0, err
    }

    // 分页查询
    offset := (page - 1) * pageSize
    if err := query.Order("created_at DESC").Offset(offset).Limit(pageSize).Find(&reports).Error; err != nil {
        return nil, 0, err
    }

    return reports, total, nil
}
```

#### 4.2.2 PDF生成服务

**文件**: `internal/infrastructure/pdf/report_generator.go`

```go
package pdf

import (
    "bytes"
    "fmt"
    "time"

    "github.com/jung-kurt/gofpdf"
    "github.com/wcharczuk/go-chart/v2"
    "github.com/xinyiis/BridgeDefectDetectionSys/internal/domain/model"
)

// ReportData 报表数据
type ReportData struct {
    Report          *model.Report
    Bridge          *model.Bridge
    Defects         []model.Defect
    DefectTypeDist  map[string]int      // 缺陷类型分布
    DefectTrend     []TrendPoint        // 缺陷趋势
    HighRiskDefects []model.Defect      // 高危缺陷列表
}

// TrendPoint 趋势数据点
type TrendPoint struct {
    Date  time.Time
    Count int
}

// ReportGenerator PDF报表生成器
type ReportGenerator struct {
    fontPath string // 字体文件路径
}

// NewReportGenerator 创建报表生成器
func NewReportGenerator(fontPath string) *ReportGenerator {
    return &ReportGenerator{fontPath: fontPath}
}

// GenerateBridgeInspectionReport 生成桥梁检测报告
func (g *ReportGenerator) GenerateBridgeInspectionReport(data *ReportData) ([]byte, error) {
    pdf := gofpdf.New("P", "mm", "A4", "")

    // 添加中文字体
    pdf.AddUTF8Font("SourceHanSans", "", g.fontPath)

    // 封面
    g.addCoverPage(pdf, data)

    // 桥梁信息
    g.addBridgeInfoSection(pdf, data)

    // 检测概览
    g.addOverviewSection(pdf, data)

    // 缺陷统计分析
    g.addStatisticsSection(pdf, data)

    // 高危缺陷列表
    g.addHighRiskDefectsSection(pdf, data)

    // 缺陷详细列表
    g.addDefectDetailsSection(pdf, data)

    // 建议和结论
    g.addConclusionSection(pdf, data)

    // 输出PDF
    var buf bytes.Buffer
    err := pdf.Output(&buf)
    if err != nil {
        return nil, err
    }

    return buf.Bytes(), nil
}

// addCoverPage 添加封面
func (g *ReportGenerator) addCoverPage(pdf *gofpdf.Fpdf, data *ReportData) {
    pdf.AddPage()
    pdf.SetFont("SourceHanSans", "", 24)
    pdf.CellFormat(0, 20, "", "", 0, "C", false, 0, "")
    pdf.Ln(50)
    pdf.CellFormat(0, 15, "桥梁缺陷检测报告", "", 1, "C", false, 0, "")
    pdf.Ln(10)

    pdf.SetFont("SourceHanSans", "", 16)
    pdf.CellFormat(0, 10, data.Bridge.BridgeName, "", 1, "C", false, 0, "")
    pdf.Ln(5)

    pdf.SetFont("SourceHanSans", "", 12)
    pdf.CellFormat(0, 8, fmt.Sprintf("检测时间：%s ~ %s",
        data.Report.StartTime.Format("2006-01-02"),
        data.Report.EndTime.Format("2006-01-02")), "", 1, "C", false, 0, "")
    pdf.Ln(30)

    pdf.SetFont("SourceHanSans", "", 10)
    pdf.CellFormat(0, 6, fmt.Sprintf("报表编号：REPORT-%d", data.Report.ID), "", 1, "C", false, 0, "")
    pdf.CellFormat(0, 6, fmt.Sprintf("生成日期：%s", time.Now().Format("2006-01-02")), "", 1, "C", false, 0, "")
}

// addBridgeInfoSection 添加桥梁信息部分
func (g *ReportGenerator) addBridgeInfoSection(pdf *gofpdf.Fpdf, data *ReportData) {
    pdf.AddPage()
    pdf.SetFont("SourceHanSans", "", 14)
    pdf.CellFormat(0, 10, "一、桥梁基本信息", "", 1, "L", false, 0, "")
    pdf.Ln(5)

    pdf.SetFont("SourceHanSans", "", 10)
    bridge := data.Bridge

    info := [][]string{
        {"桥梁名称", bridge.BridgeName},
        {"桥梁编号", bridge.BridgeCode},
        {"桥梁地址", bridge.Address},
        {"桥梁类型", bridge.BridgeType},
```

```go
// CreateReport 创建报表
func (uc *ReportUseCase) CreateReport(req *dto.CreateReportRequest, currentUser *model.User) (*dto.ReportResponse, error) {
    // 1. 参数验证
    if err := uc.validateCreateRequest(req); err != nil {
        return nil, err
    }

    // 2. 解析时间
    startTime, err := time.Parse("2006-01-02", req.StartTime)
    if err != nil {
        return nil, errors.New("开始时间格式错误")
    }
    endTime, err := time.Parse("2006-01-02", req.EndTime)
    if err != nil {
        return nil, errors.New("结束时间格式错误")
    }

    // 3. 验证桥梁所有权
    if req.BridgeID != nil {
        bridge, err := uc.bridgeService.GetBridge(*req.BridgeID, currentUser)
        if err != nil {
            return nil, errors.New("桥梁不存在或无权访问")
        }
        if !currentUser.IsAdmin() && !bridge.IsOwnedBy(currentUser.ID) {
            return nil, errors.New("无权生成该桥梁的报表")
        }
    }

    // 4. 创建报表记录（状态为generating）
    report := &model.Report{
        ReportName: req.ReportName,
        ReportType: req.ReportType,
        UserID:     req.UserID,
        BridgeID:   req.BridgeID,
        BridgeIDs:  req.BridgeIDs,
        StartTime:  startTime,
        EndTime:    endTime,
        Status:     model.ReportStatusGenerating,
    }

    if err := uc.reportService.CreateReport(report); err != nil {
        return nil, err
    }

    // 5. 异步生成PDF
    go uc.generatePDFAsync(report, currentUser)

    // 6. 返回报表信息
    return uc.toReportResponse(report), nil
}

// generatePDFAsync 异步生成PDF
func (uc *ReportUseCase) generatePDFAsync(report *model.Report, currentUser *model.User) {
    // 1. 收集报表数据
    data, err := uc.collectReportData(report, currentUser)
    if err != nil {
        report.MarkAsFailed(err.Error())
        uc.reportService.UpdateReport(report)
        return
    }

    // 2. 生成PDF
    pdfBytes, err := uc.pdfGenerator.GenerateBridgeInspectionReport(data)
    if err != nil {
        report.MarkAsFailed(fmt.Sprintf("PDF生成失败：%s", err.Error()))
        uc.reportService.UpdateReport(report)
        return
    }

    // 3. 保存PDF文件
    fileName := fmt.Sprintf("report_%d_%s.pdf", report.ID, time.Now().Format("20060102150405"))
    filePath := filepath.Join(uc.reportBaseDir, fileName)

    if err := os.WriteFile(filePath, pdfBytes, 0644); err != nil {
        report.MarkAsFailed(fmt.Sprintf("文件保存失败：%s", err.Error()))
        uc.reportService.UpdateReport(report)
        return
    }

    // 4. 更新报表状态
    totalPages := len(pdfBytes) / 4096 // 粗略估算页数
    report.MarkAsCompleted(filePath, int64(len(pdfBytes)), totalPages)
    uc.reportService.UpdateReport(report)
}

// collectReportData 收集报表数据
func (uc *ReportUseCase) collectReportData(report *model.Report, currentUser *model.User) (*pdf.ReportData, error) {
    // 1. 获取桥梁信息
    bridge, err := uc.bridgeService.GetBridge(*report.BridgeID, currentUser)
    if err != nil {
        return nil, err
    }

    // 2. 查询缺陷列表
    defects, err := uc.defectService.ListDefectsByBridgeAndTime(
        *report.BridgeID,
        report.StartTime,
        report.EndTime,
        currentUser,
    )
    if err != nil {
        return nil, err
    }

    // 3. 统计缺陷类型分布
    defectTypeDist := make(map[string]int)
    for _, defect := range defects {
        defectTypeDist[defect.DefectType]++
    }

    // 4. 获取高危缺陷
    highRiskDefects := []model.Defect{}
    for _, defect := range defects {
        if defect.Confidence >= 0.85 || defect.Area >= 0.02 {
            highRiskDefects = append(highRiskDefects, defect)
        }
    }

    // 5. 计算健康度评分
    healthScore := 100.0 - float64(len(defects)) - float64(len(highRiskDefects))*5.0
    if healthScore < 0 {
        healthScore = 0
    }

    // 6. 更新报表统计字段
    report.DefectCount = len(defects)
    report.HighRiskCount = len(highRiskDefects)
    report.HealthScore = healthScore

    // 7. 生成趋势数据（按天统计）
    defectTrend := uc.generateDefectTrend(defects, report.StartTime, report.EndTime)

    return &pdf.ReportData{
        Report:          report,
        Bridge:          bridge,
        Defects:         defects,
        DefectTypeDist:  defectTypeDist,
        DefectTrend:     defectTrend,
        HighRiskDefects: highRiskDefects,
    }, nil
}

// generateDefectTrend 生成缺陷趋势数据
func (uc *ReportUseCase) generateDefectTrend(defects []model.Defect, startTime, endTime time.Time) []pdf.TrendPoint {
    trendMap := make(map[string]int)

    for _, defect := range defects {
        dateKey := defect.DetectedAt.Format("2006-01-02")
        trendMap[dateKey]++
    }

    trend := []pdf.TrendPoint{}
    for date := startTime; !date.After(endTime); date = date.AddDate(0, 0, 1) {
        dateKey := date.Format("2006-01-02")
        trend = append(trend, pdf.TrendPoint{
            Date:  date,
            Count: trendMap[dateKey],
        })
    }

    return trend
}

// validateCreateRequest 验证创建请求
func (uc *ReportUseCase) validateCreateRequest(req *dto.CreateReportRequest) error {
    switch req.ReportType {
    case model.ReportTypeBridgeInspection:
        if req.BridgeID == nil {
            return errors.New("桥梁检测报告必须指定bridge_id")
        }
    case model.ReportTypeHealthComparison:
        if len(req.BridgeIDs) < 2 {
            return errors.New("健康对比报告至少需要2座桥梁")
        }
    }
    return nil
}
```

#### 4.4.2 中间件（ReportOwnershipRequired）

**文件**: `internal/interfaces/http/middleware/resource_ownership.go`（修改）

```go
// ReportOwnershipRequired 报表所有权验证中间件
func ReportOwnershipRequired(reportRepo repository.ReportRepository) gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 获取当前用户
        currentUser, exists := c.Get("current_user")
        if !exists {
            response.Unauthorized(c, "未登录")
            c.Abort()
            return
        }
        user := currentUser.(*model.User)

        // 2. 管理员直接放行（无需查询数据库）
        if user.IsAdmin() {
            c.Next()
            return
        }

        // 3. 获取报表ID
        reportIDStr := c.Param("id")
        reportID, err := strconv.ParseUint(reportIDStr, 10, 64)
        if err != nil {
            response.BadRequest(c, "报表ID格式错误")
            c.Abort()
            return
        }

        // 4. 查询报表
        report, err := reportRepo.FindByID(uint(reportID))
        if err != nil {
            response.NotFound(c, "报表不存在")
            c.Abort()
            return
        }

        // 5. 验证所有权
        if !report.IsOwnedBy(user.ID) {
            response.Forbidden(c, "无权访问此报表")
            c.Abort()
            return
        }

        // 6. 存入上下文（避免Handler重复查询）
        c.Set("report", report)
        c.Next()
    }
}
```
