## 1. 模块概述

### 1.1 功能定位

缺陷检测模块是桥梁缺陷检测系统的**核心业务模块**，负责图片上传、调用AI模型检测、保存检测结果、管理缺陷记录。是连接前端用户和Python AI服务的桥梁。

### 1.2 核心功能

#### 阶段1：图片上传与缺陷记录管理（本次实现）

- **图片上传检测**: 上传图片 → 保存到服务器 → 调用Python服务（Mock）→ 保存检测结果
- **缺陷列表查询**: 支持分页、按桥梁过滤、按缺陷类型过滤、权限过滤
- **缺陷详情查看**: 查看单条缺陷记录，包含原图、结果图、检测数据
- **缺陷删除**: 软删除缺陷记录（管理员或所属桥梁的拥有者）

#### 阶段2：视频流检测（后续实现）

- 实时视频流检测
- WebSocket通信
- 多帧结果聚合

### 1.3 业务流程

```
用户上传图片
    ↓
保存图片到 /uploads/images/
    ↓
创建Defect记录（状态：等待检测）
    ↓
调用Python服务检测（异步/同步）
    ↓
Python返回检测结果
    ↓
保存结果图到 /uploads/results/
    ↓
更新Defect记录（状态：完成）
    ↓
返回检测结果给用户
```

### 1.4 权限规则

| 角色 | 上传检测 | 查看缺陷 | 删除缺陷 |
|------|---------|---------|---------|
| 普通用户 | ✅ 自己的桥梁 | 仅自己桥梁的 | 仅自己桥梁的 |
| 管理员 | ✅ 所有桥梁 | 全部 | 全部 |

## 2. 功能需求分析

### 2.1 图片上传检测

#### 功能描述
用户选择桥梁、上传图片，系统调用AI模型检测缺陷，保存结果到数据库。

#### 业务规则
1. **图片格式**: 支持 jpg、jpeg、png、bmp
2. **图片大小**: 最大10MB
3. **关联桥梁**: 可选，未关联的图片仍可检测，但统计时不计入
4. **模型参数**:
   - `model_name`: 模型名称/版本（默认：yolov8）
   - `pixel_ratio`: 像素实际系数，用于将像素尺寸转换为实际尺寸（米）

### 4.3 数据库表结构

```sql
CREATE TABLE `defects` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT,
  `bridge_id` bigint unsigned NOT NULL COMMENT '所属桥梁ID',
  `defect_type` varchar(50) NOT NULL COMMENT '缺陷类型',
  `image_path` varchar(255) NOT NULL COMMENT '原始图片路径',
  `result_path` varchar(255) DEFAULT NULL COMMENT '检测结果图片路径',
  `bbox` text COMMENT '边界框坐标JSON',
  `length` decimal(10,4) DEFAULT NULL COMMENT '缺陷长度（米）',
  `width` decimal(10,4) DEFAULT NULL COMMENT '缺陷宽度（米）',
  `area` decimal(10,4) DEFAULT NULL COMMENT '缺陷面积（平方米）',
  `confidence` decimal(5,4) DEFAULT NULL COMMENT '置信度',
  `detected_at` datetime NOT NULL COMMENT '检测时间',
  `created_at` datetime DEFAULT NULL,
  `deleted_at` datetime DEFAULT NULL,
  PRIMARY KEY (`id`),
  KEY `idx_defects_bridge_id` (`bridge_id`),
  KEY `idx_defects_defect_type` (`defect_type`),
  KEY `idx_defects_detected_at` (`detected_at`),
  KEY `idx_defects_deleted_at` (`deleted_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='缺陷检测结果表';
```

**索引设计说明**：
- `PRIMARY KEY (id)`: 主键索引
- `idx_defects_bridge_id`: 查询某桥梁的缺陷、JOIN bridges表
- `idx_defects_defect_type`: 按缺陷类型筛选、统计分析
- `idx_defects_detected_at`: 时间范围查询、按时间排序
- `idx_defects_deleted_at`: GORM软删除（每次查询都使用）

**外键说明**：
- ❌ **不使用数据库外键约束**
- ✅ 通过应用层Service验证数据完整性
- ✅ 性能优先（高并发写入无外键锁开销）
- ✅ 灵活性优先（便于测试、迁移、分库）
- ✅ 现代Web应用最佳实践
```

## 6. 核心技术实现方案

### 6.1 图片上传检测流程

```go
func (uc *DetectionUseCase) UploadAndDetect(req *dto.DetectionUploadRequest, currentUser *model.User) (*dto.DetectionResponse, error) {
    startTime := time.Now()

    // 1. 验证桥梁所有权（如果指定了bridge_id）
    if req.BridgeID != nil {
        bridge, err := uc.bridgeService.GetByID(*req.BridgeID)
        if err != nil {
            return nil, errors.New("桥梁不存在")
        }
        if !currentUser.IsAdmin() && !bridge.IsOwnedBy(currentUser.ID) {
            return nil, errors.New("无权访问此桥梁")
        }
    }

    // 2. 保存上传的图片
    imagePath, err := uc.fileService.SaveImage(req.Image)
    if err != nil {
        return nil, err
    }

    // 3. 调用Python服务检测（返回多个缺陷）
    pythonResult, err := uc.pythonService.DetectDefect(imagePath, req.ModelName, req.PixelRatio)
    if err != nil {
        // 删除已上传的图片
        uc.fileService.DeleteFile(imagePath)
        return nil, err
    }

    // 4. 保存结果图（如果有）- 只有一张，包含所有缺陷标注
    var resultPath string
    if pythonResult.ResultImage != "" {
        resultPath, err = uc.fileService.SaveResultImage(pythonResult.ResultImage)
        if err != nil {
            uc.fileService.DeleteFile(imagePath)
            return nil, err
        }
    }

    // 5. 为每个检测到的缺陷创建数据库记录
    defects := make([]*model.Defect, 0, len(pythonResult.Defects))

    for _, detectedDefect := range pythonResult.Defects {
        defect := &model.Defect{
            BridgeID:   req.BridgeID,
            DefectType: detectedDefect.DefectType,
            ImagePath:  imagePath,      // 共享同一张原图
            ResultPath: resultPath,     // 共享同一张结果图
            BBox:       detectedDefect.BBoxJSON(),
            Length:     detectedDefect.Length,
            Width:      detectedDefect.Width,
            Area:       detectedDefect.Area,
            Confidence: detectedDefect.Confidence,
            DetectedAt: time.Now(),
        }

        if err := uc.defectService.CreateDefect(defect); err != nil {
            log.Printf("保存缺陷记录失败: %v", err)
            continue  // 继续处理其他缺陷
        }

        defects = append(defects, defect)
    }

    // 6. 如果一个缺陷都没保存成功，删除文件
    if len(defects) == 0 && len(pythonResult.Defects) > 0 {
        uc.fileService.DeleteFile(imagePath)
        if resultPath != "" {
            uc.fileService.DeleteFile(resultPath)
        }
        return nil, errors.New("保存缺陷记录失败")
    }

    // 7. 计算处理时间
    processingTime := time.Since(startTime).Seconds()

    // 8. 返回多个缺陷结果
    return &dto.DetectionResponse{
        TotalDefects:   len(defects),
        ImagePath:      imagePath,
        ResultPath:     resultPath,
        ProcessingTime: processingTime,
        Defects:        uc.toDefectDTOs(defects),
        DefectSummary:  uc.buildDefectSummary(defects),
    }, nil
}

// toDefectDTOs 转换为DTO列表
func (uc *DetectionUseCase) toDefectDTOs(defects []*model.Defect) []dto.DefectDTO {
    dtos := make([]dto.DefectDTO, 0, len(defects))
    for _, defect := range defects {
        dtos = append(dtos, dto.DefectDTO{
            DefectID:   defect.ID,
            DefectType: defect.DefectType,
            BBox:       parseBBox(defect.BBox),
            Length:     defect.Length,
            Width:      defect.Width,
            Area:       defect.Area,
            Confidence: defect.Confidence,
            DetectedAt: defect.DetectedAt,
        })
    }
    return dtos
}

// buildDefectSummary 构建缺陷统计
func (uc *DetectionUseCase) buildDefectSummary(defects []*model.Defect) map[string]int {
    summary := make(map[string]int)
    for _, defect := range defects {
        summary[defect.DefectType]++
    }
    return summary
}

#### Service层权限过滤

```go
// ListDefects 获取缺陷列表（带权限过滤）
func (s *DefectService) ListDefects(currentUser *model.User, filters *DefectFilters, page, pageSize int) ([]model.Defect, int64, error) {
    query := s.db.Model(&model.Defect{})

    // 1. 权限过滤
    if !currentUser.IsAdmin() {
        // 普通用户只能看自己桥梁的缺陷
        // 需要JOIN bridges表来检查user_id
        query = query.Joins("JOIN bridges ON bridges.id = defects.bridge_id").
                Where("bridges.user_id = ?", currentUser.ID)
    }

    // 2. 其他过滤条件
    if filters.BridgeID != nil {
        query = query.Where("defects.bridge_id = ?", *filters.BridgeID)
    }
    if filters.DefectType != "" {
        query = query.Where("defects.defect_type = ?", filters.DefectType)
    }
    if !filters.StartDate.IsZero() {
        query = query.Where("defects.detected_at >= ?", filters.StartDate)
    }
    if !filters.EndDate.IsZero() {
        query = query.Where("defects.detected_at <= ?", filters.EndDate)
    }

    // 3. 统计总数
    var total int64
    query.Count(&total)

    // 4. 分页查询
    var defects []model.Defect
    offset := (page - 1) * pageSize
    err := query.Preload("Bridge").
            Offset(offset).
            Limit(pageSize).
            Order("detected_at DESC").
            Find(&defects).Error

    return defects, total, err
}

```go
// DefectOwnershipRequired 验证用户是否有权访问缺陷记录
func DefectOwnershipRequired(defectRepo repository.DefectRepository, bridgeRepo repository.BridgeRepository) gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 获取当前用户
        currentUser, exists := c.Get("current_user")
        if !exists {
            response.Unauthorized(c)
            c.Abort()
            return
        }
        user := currentUser.(*model.User)

        // 2. 管理员直接放行
        if user.IsAdmin() {
            c.Next()
            return
        }

        // 3. 获取缺陷ID
        defectIDStr := c.Param("id")
        defectID, err := strconv.ParseUint(defectIDStr, 10, 64)
        if err != nil {
            response.BadRequest(c, "无效的缺陷ID")
            c.Abort()
            return
        }

        // 4. 查询缺陷
        defect, err := defectRepo.FindByID(uint(defectID))
        if err != nil || defect == nil {
            response.NotFound(c, "缺陷不存在")
            c.Abort()
            return
        }

        // 5. 查询关联的桥梁，验证所有权
        bridge, err := bridgeRepo.FindByID(defect.BridgeID)
        if err != nil || bridge == nil {
            response.NotFound(c, "关联桥梁不存在")
            c.Abort()
            return
        }

        if !bridge.IsOwnedBy(user.ID) {
            response.Forbidden(c)
            c.Abort()
            return
        }

        // 6. 存入上下文
        c.Set("defect", defect)
        c.Next()
    }
}

### 6.3 缺陷删除与文件清理

```go
// DeleteDefect 删除缺陷（软删除+文件清理）
func (s *DefectService) DeleteDefect(defectID uint, currentUser *model.User) error {
    // 1. 查询缺陷
    defect, err := s.defectRepo.FindByID(defectID)
    if err != nil {
        return err
    }
    if defect == nil {
        return errors.New("缺陷不存在")
    }

    // 2. 验证权限（查询关联桥梁）
    bridge, err := s.bridgeRepo.FindByID(defect.BridgeID)
    if err != nil {
        return err
    }
    if !currentUser.IsAdmin() && !bridge.IsOwnedBy(currentUser.ID) {
        return errors.New("权限不足")
    }

    // 3. 软删除（无需事务，只是标记deleted_at）
    if err := s.db.Delete(defect).Error; err != nil {
        return err
    }

    // 4. 删除文件（异步，失败不影响软删除结果）
    go func() {
        if defect.ImagePath != "" {
            s.fileService.DeleteFile(defect.ImagePath)
        }
        if defect.ResultPath != "" {
            s.fileService.DeleteFile(defect.ResultPath)
        }
    }()

    return nil
}

// HTTPPythonService Python服务HTTP实现
type HTTPPythonService struct {
    baseURL string
    client  *http.Client
}

func NewHTTPPythonService(baseURL string) *HTTPPythonService {
    return &HTTPPythonService{
        baseURL: baseURL,
        client: &http.Client{
            Timeout: 30 * time.Second,
        },
    }
}

func (s *HTTPPythonService) DetectDefect(imagePath, modelName string, pixelRatio float64) (*DetectionResult, error) {
    // 1. 构建请求
    requestBody := map[string]interface{}{
        "image_path":  imagePath,
        "model_name":  modelName,
        "pixel_ratio": pixelRatio,
    }

    jsonData, err := json.Marshal(requestBody)
    if err != nil {
        return nil, err
    }

    // 2. 发送HTTP请求
    url := fmt.Sprintf("%s/api/detect", s.baseURL)
    resp, err := s.client.Post(url, "application/json", bytes.NewBuffer(jsonData))
    if err != nil {
        return nil, fmt.Errorf("调用Python服务失败: %w", err)
    }
    defer resp.Body.Close()

    // 3. 解析响应
    if resp.StatusCode != http.StatusOK {
        return nil, fmt.Errorf("Python服务返回错误: %d", resp.StatusCode)
    }

    var result DetectionResult
    if err := json.NewDecoder(resp.Body).Decode(&result); err != nil {
        return nil, err
    }

    return &result, nil
}
