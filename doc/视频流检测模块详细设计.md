## 1. 模块概述

### 1.1 功能描述

视频流检测模块是桥梁缺陷检测系统的核心功能，实现从视频源（无人机/摄像头）实时读取视频流，通过AI算法检测桥梁缺陷，并将检测结果实时推送给前端用户。

### 1.2 核心功能

- ✅ **多种视频源支持**：无人机RTSP流、电脑摄像头、本地视频文件、测试流
- ✅ **实时AI检测**：逐帧检测桥梁缺陷（裂缝、剥落、锈蚀等）
- ✅ **WebSocket实时推送**：检测结果实时推送到前端
- ✅ **会话管理**：记录检测任务的完整生命周期
- ✅ **权限控制**：用户只能使用自己的设备和桥梁
- ✅ **统计分析**：实时统计帧数、缺陷数、检测时长

### 1.3 技术选型

| 技术栈 | 选型 | 说明 |
|--------|------|------|
| **后端语言** | Go 1.21+ | 高并发、协程支持 |
| **Web框架** | Gin | 轻量级、高性能 |
| **WebSocket** | gorilla/websocket | Go标准WebSocket库 |
| **视频处理** | gocv (OpenCV Go绑定) | 跨平台视频流处理 |
| **AI服务** | Python + Flask | YOLO/SAM推理 |
| **数据库** | MySQL 8.0 | 关系型数据库 |
| **前端框架** | Vue 3 + Element Plus | 响应式UI |

## 2. 业务场景分析

### 2.1 真实场景 vs 开发场景

#### **真实场景：无人机巡检**

```
场景描述：
工程师操控无人机飞到桥梁附近，启动视频流检测，
AI实时分析视频画面，发现缺陷后自动保存记录。

特点：
- 设备：大疆 Mavic 3 等专业无人机
- 视频源：RTSP流（动态IP，临时连接）
- 使用时长：10-30分钟/次
- 使用频率：每周1-2次
```

#### **开发场景：电脑摄像头模拟**

```
场景描述：
开发阶段没有真实无人机，使用电脑摄像头模拟视频流，
拍摄桥梁图片/视频进行AI检测测试。

特点：
- 设备：笔记本内置摄像头
- 视频源：设备ID（如：0、1）
- 使用时长：不限
- 使用频率：频繁测试
```

#### **方案设计原则**

> **核心思想**：系统不关心视频来源，统一抽象为 `VideoSource` 接口，
> 支持多种视频源类型，开发时用摄像头，部署时无缝切换到无人机。

### 5.2 VideoSource 实现

#### **RTSPSource（无人机）**

```go
type RTSPSource struct {
    url string
    cap *gocv.VideoCapture
}

func NewRTSPSource(url string) *RTSPSource {
    return &RTSPSource{url: url}
}

func (s *RTSPSource) Open() error {
    var err error
    s.cap, err = gocv.VideoCaptureFile(s.url)
    if err != nil {
        return fmt.Errorf("无法打开RTSP流: %w", err)
    }
    return nil
}

func (s *RTSPSource) Read() (Frame, error) {
    img := gocv.NewMat()
    defer img.Close()

    if ok := s.cap.Read(&img); !ok {
        return Frame{}, errors.New("读取帧失败")
    }

    buf, _ := gocv.IMEncode(".jpg", img)
    return Frame{
        Data:      buf.GetBytes(),
        Width:     img.Cols(),
        Height:    img.Rows(),
        Timestamp: time.Now(),
    }, nil
}

func (s *RTSPSource) GetType() string {
    return "rtsp"
}
```

## 6. 功能实现方案

### 6.1 StreamHandler（核心）

```go
type StreamHandler struct {
    droneRepo          repository.DroneRepository
    sessionRepo        repository.DetectionSessionRepository
    defectRepo         repository.DefectRepository
    pythonService      service.PythonService
    videoSourceFactory *VideoSourceFactory
    activeSessions     map[string]*ActiveSession
    mu                 sync.RWMutex
}

type ActiveSession struct {
    SessionID   string
    VideoSource VideoSource
    Context     context.Context
    CancelFunc  context.CancelFunc
    FrameCount  int
    DefectCount int
}

func (h *StreamHandler) HandleStreamDetection(c *gin.Context) {
    // 1. 升级WebSocket
    conn, _ := upgrader.Upgrade(c.Writer, c.Request, nil)
    defer conn.Close()

    // 2. 接收启动消息
    var msg StartStreamMessage
    conn.ReadJSON(&msg)

    // 3. 验证权限
    // ...

    // 4. 创建视频源
    videoSource, _ := h.videoSourceFactory.CreateVideoSource(drone)
    videoSource.Open()

    // 5. 创建会话
    session := &model.DetectionSession{
        SessionID: uuid.New().String(),
        // ...
    }
    h.sessionRepo.Create(session)

    // 6. 启动检测协程
    ctx, cancel := context.WithCancel(context.Background())
    go h.runDetectionPipeline(ctx, conn, session, videoSource)

    // 7. 等待停止信号
    h.waitForStopSignal(conn, session.SessionID)
}
```

### 6.2 检测流水线

```go
func (h *StreamHandler) runDetectionPipeline(
    ctx context.Context,
    conn *websocket.Conn,
    session *model.DetectionSession,
    videoSource VideoSource,
) {
    defer videoSource.Close()

    ticker := time.NewTicker(500 * time.Millisecond)
    defer ticker.Stop()

    for {
        select {
        case <-ctx.Done():
            return
        case <-ticker.C:
            // 处理一帧
            frame, _ := videoSource.Read()

            // 保存临时图片
            tempPath := fmt.Sprintf("/tmp/frame_%d.jpg", frameCount)
            ioutil.WriteFile(tempPath, frame.Data, 0644)

            // AI检测
            result, _ := h.pythonService.DetectDefect(tempPath, ...)

            // 保存缺陷
            if len(result.Defects) > 0 {
                h.defectRepo.Create(...)
            }

            // 推送结果
            h.sendDetectionResult(conn, result)
        }
    }
}
```
