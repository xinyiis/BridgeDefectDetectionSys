## 🚀 快速开始

### 第一步：启动后端服务

```bash

bash start_backend.sh

# 方法2：手动启动
cd src/backend
go run cmd/server/main.go
```

等待看到以下输出：
```

监听地址: :8080
```

### 第二步：运行端到端测试

**在另一个终端窗口运行**：

```bash

```

## 🔧 环境要求

### 必需工具

```bash
# Ubuntu/Debian
sudo apt-get install curl jq mysql-client

# macOS
brew install curl jq mysql-client
```

### 数据库

确保 MySQL 正在运行：
```bash
sudo systemctl status mysql
```

数据库配置：
- 用户名: `root`
- 密码: `123456`
- 数据库: `bridge_detection`
- 端口: `3306`

## 🐛 故障排除

### 问题1：登录失败 (401错误)

**原因**：数据库中没有测试用户

**解决**：运行端到端测试，它会自动创建用户

### 问题2：测试脚本报错 "jq: command not found"

**解决**：
```bash
sudo apt-get install jq  # Ubuntu
brew install jq          # macOS
```

### 问题3：MySQL 连接失败

**解决**：
```bash
# 检查 MySQL 状态
sudo systemctl status mysql

# 启动 MySQL
sudo systemctl start mysql

# 测试连接
mysql -uroot -p123456 -e "SHOW DATABASES;"
```

### 问题4：服务启动失败

**可能原因**：
1. 端口 8080 被占用
2. 配置文件错误
3. Go 版本不兼容

**检查方法**：
```bash
# 检查端口占用
lsof -i :8080

# 检查 Go 版本
go version  # 需要 1.25+

# 查看详细错误
cd src/backend
go run cmd/server/main.go
```
