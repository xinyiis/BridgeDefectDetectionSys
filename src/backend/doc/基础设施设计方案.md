- **项目名称**: 桥梁缺陷检测系统
- **文档类型**: 基础设施设计方案

- **设计原则**: 够用就好，快速开发，避免过度设计

## 一、设计目标

### 1.1 核心目标

1. **快速开发**: 减少学习成本，使用成熟简单的技术栈
2. **满足需求**: 支撑所有业务功能（用户管理、桥梁管理、检测识别等）
3. **可维护**: 代码结构清晰，便于团队协作
4. **可扩展**: 预留扩展空间，但不过度设计

### 1.2 非目标（避免过度设计）

- ❌ 不追求高并发性能（比赛项目，用户量不大）
- ❌ 不做复杂的分布式架构（单体应用足够）
- ❌ 不引入过多第三方库（降低复杂度）
- ❌ 不做过度抽象（保持简单直接）

```go
func InitDatabase(config *Config) *gorm.DB {
    // 1. 打开MySQL连接
    db, err := gorm.Open(mysql.Open(config.Database.DSN), &gorm.Config{
        // 禁用默认事务，提高性能
        SkipDefaultTransaction: true,
        // 命名策略：表名单数
        NamingStrategy: schema.NamingStrategy{
            SingularTable: true,
        },
    })

    if err != nil {
        log.Fatal("数据库连接失败:", err)
    }

    // 2. 自动迁移
    AutoMigrate(db)

    // 3. 配置连接池
    sqlDB, _ := db.DB()
    sqlDB.SetMaxIdleConns(10)
    sqlDB.SetMaxOpenConns(100)

    return db
}

func AutoMigrate(db *gorm.DB) {
```

##### 2. 认证中间件
```go
func AuthRequired(db *gorm.DB) gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 获取Session
        session := sessions.Default(c)
        userID := session.Get("user_id")

        if userID == nil {
            c.JSON(401, gin.H{"error": "未登录"})
            c.Abort()
            return
        }

        // 2. 从数据库加载用户信息
        var user model.User
        if err := db.First(&user, userID).Error; err != nil {
            c.JSON(401, gin.H{"error": "用户不存在"})
            c.Abort()
            return
        }

        // 3. 将用户信息存入Context
        c.Set("current_user", &user)
        c.Next()
    }
}
```

##### 3. 管理员权限中间件
```go
func AdminRequired() gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 从Context获取当前用户
        userInterface, exists := c.Get("current_user")
        if !exists {
            c.JSON(401, gin.H{"error": "未登录"})
            c.Abort()
            return
        }

        user := userInterface.(*model.User)

        // 2. 检查是否为管理员
        if user.Role != "admin" {
            c.JSON(403, gin.H{"error": "权限不足"})
            c.Abort()
            return
        }

        c.Next()
    }
}
```

##### 4. 资源权限检查（可选）
```go
// 检查用户是否有权访问某个资源（如桥梁、无人机）
func CheckResourceOwnership(db *gorm.DB, resourceType string) gin.HandlerFunc {
    return func(c *gin.Context) {
        user := c.MustGet("current_user").(*model.User)

        // 管理员跳过检查
        if user.Role == "admin" {
            c.Next()
            return
        }

        // 普通用户检查资源所有权
        resourceID := c.Param("id")

        switch resourceType {

                c.Abort()
                return
            }
            if bridge.UserID != user.ID {
                c.JSON(403, gin.H{"error": "无权访问此桥梁"})
                c.Abort()
                return
            }
        // 其他资源类型...
        }

        c.Next()
    }
```

### 5.4 路由管理 (interfaces/http/router/router.go)

#### 职责
- 注册所有路由
- 绑定中间件
- 路由分组管理
- 静态文件服务

#### 核心结构
```go
func SetupRouter(db *gorm.DB, config *Config) *gin.Engine {
    r := gin.Default()

    // 1. 全局中间件
    r.Use(middleware.CORSMiddleware())

    // 2. Session配置
    store := cookie.NewStore([]byte("your-secret-key"))
    r.Use(sessions.Sessions("bridge_session", store))

    // 3. 静态文件服务（检测结果图片）
    r.Static("/uploads", "./uploads")

    // 4. API路由组
    api := r.Group("/api")

    // 5. 公开路由（无需登录）
    registerPublicRoutes(api, db)

    // 6. 认证路由（需要登录）
    auth := api.Group("/")
    auth.Use(middleware.AuthRequired(db))
    registerAuthRoutes(auth, db)

    // 7. 管理员路由（需要管理员权限）
    admin := api.Group("/admin")
    admin.Use(middleware.AuthRequired(db))
    admin.Use(middleware.AdminRequired())
    registerAdminRoutes(admin, db)

    return r
}
```

#### 路由分组设计
```go
// 公开路由
func registerPublicRoutes(r *gin.RouterGroup, db *gorm.DB) {
    r.POST("/register", handler.Register(db))
    r.POST("/login", handler.Login(db))
}

// 认证用户路由
func registerAuthRoutes(r *gin.RouterGroup, db *gorm.DB) {
    // 用户相关
    r.POST("/logout", handler.Logout)
    r.GET("/user/info", handler.GetUserInfo)
```

## 六、依赖关系

### 6.1 模块依赖图

```
main.go
  │
  ├─► config.LoadConfig()
  │     └─► 读取 config.yaml
  │
  ├─► persistence.InitDatabase(config)
  │     ├─► 连接MySQL
  │     └─► AutoMigrate(models)
  │
  └─► router.SetupRouter(db, config)
        ├─► middleware.CORSMiddleware()
        ├─► middleware.AuthRequired(db)
        ├─► middleware.AdminRequired()
        └─► handler.*（所有Handler）
              └─► usecase.*（所有UseCase）
                    ├─► repository.*（数据访问）
                    └─► external.*（外部服务）
```

### 6.2 依赖注入原则

```
高层模块（Handler）不依赖低层模块（Repository）
      │
      ▼
两者都依赖抽象（Interface）
      │
      ▼
通过构造函数注入具体实现
```

**示例**：
```go
// 定义接口（domain层）
type UserRepository interface {
    Create(user *User) error
    FindByID(id uint) (*User, error)
    FindByUsername(username string) (*User, error)
}

// 实现接口（infrastructure层）
type userRepositoryImpl struct {
    db *gorm.DB
}

func NewUserRepository(db *gorm.DB) UserRepository {
    return &userRepositoryImpl{db: db}
}

// 使用接口（handler层）
type UserHandler struct {
    repo UserRepository
}

func NewUserHandler(repo UserRepository) *UserHandler {
    return &UserHandler{repo: repo}
}
```

## 九、安全考虑

### 9.1 密码安全
- 使用 bcrypt 加密存储
- 加密成本因子：10-12

```go
import "golang.org/x/crypto/bcrypt"

func HashPassword(password string) (string, error) {
    bytes, err := bcrypt.GenerateFromPassword([]byte(password), 10)
    return string(bytes), err
}

func CheckPassword(password, hash string) bool {
    err := bcrypt.CompareHashAndPassword([]byte(hash), []byte(password))
    return err == nil
}
```

### 9.2 Session安全
- 使用随机密钥
- 生产环境密钥必须从环境变量读取
- 设置合理的过期时间

### 9.3 SQL注入防护
- 使用GORM的参数化查询
- 禁止拼接SQL字符串

### 9.4 文件上传安全
- 验证文件类型
- 限制文件大小
- 重命名上传文件（使用UUID）

```go
func SaveUploadedFile(c *gin.Context, file *multipart.FileHeader) (string, error) {
    // 1. 验证文件类型
    ext := filepath.Ext(file.Filename)
    allowedExts := []string{".jpg", ".jpeg", ".png"}
    if !contains(allowedExts, ext) {
        return "", errors.New("不支持的文件类型")
    }

    // 2. 验证文件大小（10MB）
    if file.Size > 10*1024*1024 {
        return "", errors.New("文件过大")
    }

    // 3. 生成新文件名
    newFilename := uuid.New().String() + ext
    savePath := filepath.Join(config.Upload.ImageDir, newFilename)

    // 4. 保存文件
    if err := c.SaveUploadedFile(file, savePath); err != nil {
        return "", err
    }

    return savePath, nil
}
```
